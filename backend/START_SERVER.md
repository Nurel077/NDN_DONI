# Как запустить сервер

## Быстрый запуск

1. Откройте терминал в папке `backend`

2. Запустите сервер:
   ```bash
   python manage.py runserver
   ```

3. Откройте в браузере:
   - Главная страница API: http://127.0.0.1:8000/
   - Админка: http://127.0.0.1:8000/admin/
   - API игры: http://127.0.0.1:8000/api/games/

## Если сервер не запускается

### Проблема: ModuleNotFoundError для MySQL

**Решение:** Проект настроен на использование SQLite по умолчанию (не требует MySQL).

Если хотите использовать MySQL:
1. Установите pymysql: `pip install pymysql`
2. Файл `ndnstore/__init__.py` уже настроен для автоматического использования pymysql

### Проблема: Ошибки миграций

Выполните:
```bash
python manage.py makemigrations
python manage.py migrate
```

### Проблема: Нет суперпользователя для админки

Создайте:
```bash
python manage.py createsuperuser
```

## Проверка работы

После запуска сервера проверьте:

1. **Главная страница**: http://127.0.0.1:8000/
   - Должна вернуть JSON с информацией об API

2. **API игры**: http://127.0.0.1:8000/api/games/
   - Должна вернуть список игр (может быть пустым, если игры не загружены)

3. **Админка**: http://127.0.0.1:8000/admin/
   - Должна открыться страница входа

## Загрузка начальных данных

После создания суперпользователя и входа в админку:
- Добавьте игры вручную через админку
- Или выполните команду: `python manage.py load_games`

## Остановка сервера

Нажмите `Ctrl+C` в терминале, где запущен сервер.
