# 📊 Проверка базы данных SQLite

## ✅ Подтверждение: Все данные сохраняются в SQLite

### Расположение базы данных
- **Файл:** `backend/db.sqlite3`
- **Тип:** SQLite 3
- **Размер:** Автоматически увеличивается при добавлении данных

### Настройки в settings.py
```python
DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.sqlite3',
        'NAME': BASE_DIR / 'db.sqlite3',
    }
}
```

### Таблицы в базе данных

1. **auth_user** - Пользователи Django
   - id, username, email, password, date_joined, is_active, etc.

2. **user_profiles** - Профили пользователей
   - user_id, games_played, last_login_time, created_at, is_active

3. **user_settings** - Настройки пользователей
   - user_id, language, theme, email_notifications, game_notifications, news_notifications

4. **user_sessions** - Сессии пользователей (старая система)
   - user_id, session_token, created_at, expires_at, is_active, remember_me

5. **games** - Игры
   - id, name, description, game_type, game_url, icon, is_active, created_at

6. **game_sessions** - Игровые сессии
   - user_id, game_id, started_at, ended_at, duration_seconds

7. **django_session** - Django сессии (текущая система авторизации)
   - session_key, session_data, expire_date

### Как проверить данные

#### 1. Через Django Admin
```
http://127.0.0.1:8000/admin/
Логин: Xx__xX
Пароль: 12345
```

#### 2. Через Python
```python
python manage.py shell
>>> from django.contrib.auth.models import User
>>> User.objects.all()
>>> from api.models import UserProfile
>>> UserProfile.objects.all()
```

#### 3. Через SQLite Browser
1. Скачайте DB Browser for SQLite
2. Откройте `backend/db.sqlite3`
3. Просматривайте все таблицы

#### 4. Через командную строку
```bash
sqlite3 backend/db.sqlite3
.tables
SELECT * FROM auth_user;
.quit
```

### Резервное копирование

**Создание копии:**
```bash
copy backend\db.sqlite3 backend\db.sqlite3.backup
```

**Восстановление:**
```bash
copy backend\db.sqlite3.backup backend\db.sqlite3
```

### Важно!

✅ Все данные автоматически сохраняются в SQLite  
✅ База данных находится в `backend/db.sqlite3`  
✅ Не нужно настраивать MySQL или другую БД  
✅ Все операции (регистрация, вход, настройки, игры) сохраняются в SQLite
