2. Запустите сервер:
   python manage.py runserver

3. Откройте в браузере:
   http://127.0.0.1:8000/

═══════════════════════════════════════════════════════════════
  API ENDPOINTS
═══════════════════════════════════════════════════════════════

Все API доступны по адресу /api/:

POST   /api/register/          - Регистрация
POST   /api/login/              - Вход
POST   /api/logout/             - Выход
GET    /api/validate-session/   - Проверка сессии
GET    /api/profile/            - Профиль
PUT    /api/settings/            - Настройки
POST   /api/change-password/    - Смена пароля
POST   /api/change-username/    - Смена имени
POST   /api/delete-account/     - Удаление аккаунта
GET    /api/games/              - Список игр
POST   /api/start-game/         - Запуск игры
