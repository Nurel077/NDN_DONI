═══════════════════════════════════════════════════════════════
  КАК РАБОТАЕТ
═══════════════════════════════════════════════════════════════

1. Регистрация:
   - Создается пользователь через User.objects.create_user()
   - Автоматически логинится через login(request, user)
   - Django сессия создается автоматически

2. Вход:
   - Проверка через authenticate()
   - Логин через login(request, user)
   - Настройка длительности сессии (1 день или 30 дней)

3. Проверка авторизации:
   - request.user.is_authenticated
   - Django автоматически проверяет сессию

4. Выход:
   - logout(request) - удаляет сессию

5. CSRF защита:
   - Токен получается из cookies
   - Отправляется в заголовке X-CSRFToken
   - Автоматическое обновление при ошибке 403

  API ENDPOINTS

POST   /api/start-game/         - Запуск игры (IsAuthenticated)
